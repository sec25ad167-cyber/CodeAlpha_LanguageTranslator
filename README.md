CodeAlpha Language Translator

Overview

A web-based language translation application developed using Python, Streamlit, and Deep Translator. The application allows users to translate text into multiple languages through a simple and interactive interface.

Features

- Translate text into multiple languages
- User-friendly Streamlit interface
- Automatic language detection
- Fast and accurate translations

Technologies Used

- Python
- Streamlit
- Deep Translator

Installation

pip install streamlit
pip install deep-translator

Run the Application

streamlit run app.py
